# Python-library voor Elecfreaks Cutebot

Python library voor het gebruik van de [Cutebot](https://www.elecfreaks.com/learn-en/microbitKit/smart_cutebot/cutebot_car.html) in de [micro:bit Python Editor](https://python.microbit.org/).

Forked from [elecfreaks/EF_Produce_MicroPython](https://github.com/elecfreaks/EF_Produce_MicroPython)

## Gebruik

1. Project > Open > cutebot.py
2. `import cutebot`
